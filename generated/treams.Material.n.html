treams.Material.n¶

property Material.n¶

Refractive index.

The refractive index is defined by \(n = \sqrt{\epsilon \mu}\), with an enforced non-negative imaginary part.

Note

The refractive index returned is independent from the chirality parameter. For an alternative definition of the refractive index, see also Material.nmp().

Returns:

complex
